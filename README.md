# Quiz App

A Kahoot-style interactive quiz application built with Flutter and Firebase (FREE tier only).

## 🆓 FREE SERVICES ONLY

**IMPORTANT**: This project uses ONLY free tier services. No paid APIs or cloud services are used.

### Technology Stack (All Free)
- **Flutter**: Open source cross-platform framework
- **Firebase Free Tier**: 
  - Authentication (unlimited users)
  - Firestore (10GB storage, 20K writes/day)
  - Storage (5GB, 1GB/day download)
  - Hosting (10GB/month)
- **GitHub Actions**: Free tier CI/CD (2000 minutes/month)

### NOT Using (Paid Services)
- ❌ Cloud Functions - All logic implemented in Flutter app
- ❌ Firebase ML/AI services
- ❌ Third-party paid APIs
- ❌ Premium cloud services

## Getting Started

This project is a Kahoot-style quiz application following Clean Architecture and TDD principles.

### Prerequisites
- Flutter SDK 3.0+
- Dart SDK
- Firebase account (free)
- Android Studio / VS Code

### Quick Start
```bash
# Install dependencies
flutter pub get

# Run the app
flutter run

# Run tests
flutter test
```
